# Advanced Project Data Analysis
This repository contains a collection of SQL-based data analysis techniques applied to structured datasets. The goal of this project is to demonstrate practical skills in data analysis, focusing on extracting insights, identifying trends, and supporting decision-making using SQL.

---
## Project Structure
The repository is organized into multiple SQL scripts, each focusing on a specific analytical approach:

- **changes-over-time.sql**
Analyzes how key metrics evolve over time (trend analysis).
-**cumulative-analysis.sql**
Calculates running totals and cumulative metrics to track growth and performance over time.
- **data-segmentation.sql**
Segments data into meaningful groups (e.g., customers, products) for deeper insights.
- **part-to-whole analysis.sql**
Examines proportions and contributions of individual parts to the total.
- **performance-analysis.sql**
Evaluates performance metrics such as sales, efficiency, or KPIs.
